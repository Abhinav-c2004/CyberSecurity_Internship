# Python Task 08 - Smart File Organizer

## Smart File Organizer

This program automatically organizes files into different folders based on their file extensions. It also provides file statistics, search functionality, duplicate file detection, and generates a text report.

### Output

#### Directory Selection & File Scanning

The user enters the folder path, and the program scans the selected directory to display all detected files.

![Directory Selection & File Scanning](screenshots/Screenshot%202026-06-26%20151522.png)

---

#### Automatic File Organization

Automatically creates category folders and moves files into their respective folders.
