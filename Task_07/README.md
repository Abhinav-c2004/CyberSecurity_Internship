# Python Task 07 - Employee Management System

## Employee Management System

This program is a menu-driven Employee Management System built using Python and Object-Oriented Programming (OOP). It allows users to manage employee records, calculate salary statistics, and export employee data to a CSV file.

### Output

#### Add Employee

This option allows the user to add a new employee by entering the employee ID, name, department, designation, and salary.

![Add Employee](screenshots/Screenshot%202026-06-26%20142030.png)

---

#### View Employees

Displays all employee records stored in the system.

![View Employees](screenshots/Screenshot%202026-06-26%20142712.png)

---

#### Search Employee

Searches for an employee using their Employee ID.

![Search Employee](screenshots/Screenshot%202026-06-26%20142822%20-%20Copy.png)

---

#### Update Employee

Updates the details of an existing employee.

![Update Employee](screenshots/Screenshot%202026-06-26%20142839%20-%20Copy.png)

---

#### Delete Employee

Deletes an employee record using the Employee ID.

![Delete Employee](screenshots/Screenshot%202026-06-26%20142849.png)

---

#### Salary Statistics

Displays:

* Highest Salary
* Lowest Salary
* Average Salary
* Total Employees
